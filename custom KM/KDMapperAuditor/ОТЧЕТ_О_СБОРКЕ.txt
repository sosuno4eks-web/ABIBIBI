ФУНКЦИОНАЛЬНОСТЬ
----------------

7 ТЕСТОВ БЕЗОПАСНОСТИ:

[1] String Scan
    • Сканирует загруженные модули ядра
    • Ищет 20+ подозрительных строк
    • Проверяет 10+ имен уязвимых драйверов
    • Метод: NtQuerySystemInformation

[2] MmUnloadedDrivers Check
    • Проверяет список выгруженных драйверов
    • Сканирует реестр на записи сервисов
    • Ищет следы уязвимых драйверов
    • Метод: Перечисление реестра

[3] PiDDBCache Verification
    • Проверяет базу данных драйверов
    • Сканирует System32\drivers на файлы
    • Проверяет записи Code Integrity
    • Метод: Файловая система + реестр

[4] Event Log Scan
    • Сканирует журнал событий Windows
    • Проверяет последние 1000 событий
    • Ищет Event ID: 7045, 7036, 219
    • Метод: OpenEventLog + ReadEventLog

[5] IOCTL Verification
    • Тестирует стандартные IOCTL коды
    • Проверяет 8+ имен устройств
    • Пытается открыть устройства
    • Метод: CreateFile + DeviceIoControl

[6] Handle Enumeration
    • Перечисляет процессы
    • Проверяет подозрительные хэндлы
    • Ищет хэндлы к драйверам
    • Метод: CreateToolhelp32Snapshot

[7] Signature Check
    • Проверяет статус тестовой подписи
    • Читает конфигурацию Code Integrity
    • Определяет подозрительные настройки
    • Метод: Реестр CI\Config
