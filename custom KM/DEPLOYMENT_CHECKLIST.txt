========================================
  TROUBLESHOOTING
========================================

IF DRIVER FAILS TO LOAD:
  - Run as Administrator
  - Disable Secure Boot (BIOS)
  - Disable Driver Signature Enforcement
  - Check Windows Event Viewer

IF CLIENT SHOWS PROCESS_NOT_FOUND:
  - Launch Arena Breakout Infinite
  - Wait for main menu
  - Client will detect automatically

IF CLIENT SHOWS DRIVER_NOT_RESPONDING:
  - Reload driver: rtkaudio.exe RtkAudioService.sys
  - Check driver loaded successfully
  - Check Windows Event Viewer

IF CLIENT SHOWS GWORLD_INVALID:
  - Wait for game to fully load
  - Reach main menu
  - If still fails, offsets may need updating

IF OVERLAY NOT VISIBLE:
  - Close other overlays (Discord, Steam)
  - Run client as Administrator
  - Check DirectX 11 installed
