================================================================================
                        COMMON ISSUES
================================================================================

ISSUE: "Debugger detected"
FIX: Close all debuggers, Process Monitor, Process Explorer

ISSUE: Driver not loading (0xC0000603)
FIX: Disable Vulnerable Driver Blocklist (see above), reboot

ISSUE: "Device is already in use"
FIX: Reboot computer, delete driver from %temp%

ISSUE: Still detected by ACE
FIX: 
  1. Run patch_driver_resource.py to patch binary driver
  2. Change IOCTL codes in source
  3. Use different vulnerable driver
